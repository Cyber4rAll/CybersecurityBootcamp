L4 Network Security

Module 1: Understand Computer Networking

A network is simply two or more computers linked together to share data, information or resources.

To properly establish secure data communications, it is important to explore all of the technologies involved in computer communications.

Types of Networks

There are two basic types of networks:

  • Local area network (LAN) - A network typically spanning a single floor or building. This is commonly a limited geographical area.

  • Wide area network (WAN) - The term usually assigned to the long-distance connections between geographically remote networks.

Network Devices

Hubs & Switches

  • Hubs are used to connect multiple devices in a network. They’re less likely to be seen in business or corporate networks than in home networks. Hubs are wired devices and are not as smart as switches or routers.

  • You might consider using a switch, or what is also known as an intelligent hub. Switches are wired devices that know the addresses of the devices connected to them and route traffic to that port/device rather than retransmitting to all devices. Offering greater efficiency for traffic delivery and improving the overall throughput of data, switches are smarter than hubs, but not as smart as routers.

Network Devices (Cont.)

Routers & Firewalls

  • Routers are devices that act as a gateway between two networks. Routers use software-based rules to determine the best path for data transmission and are commonly used to interconnect different networks, such as a LAN to a WAN, or the internet.

  • Firewalls are a device or software program that monitors network traffic and filters it. They can be set up to deny traffic from a specific IP address or to allow traffic only from specific IP addresses. Firewalls are a key security device used to control access to the internal network from outside the network. It can be a software program or a hardware device.

Network Design

A network can be designed as a client-server or peer-to-peer network.

  • Client-server network - a network where the client computers are dependent on a server computer to access and use network resources.

  • Peer-to-peer network - In this network, all computers are equal and can share resources with each other without the need for a central server.

Module 2: Network Topologies

The way a network is laid out is called a network topology. It can be either a physical or logical layout. There are many types of topologies.

Bus & Ring Topologies

  • Bus - devices are connected in a line. All messages are transmitted to all devices on the network. It’s an inexpensive topology, but it can be difficult to manage and reconfigure.

  • Ring - In this topology, devices are connected in a circle and messages are passed from one device to the next in order. The main benefit is that it is easy to set up. A main disadvantage is that if one device fails, the whole network can go down.

Star & Mesh Topologies

  • Star - In this topology, all devices are connected to a central device. The main benefit is that it is the most reliable network topology because if one device fails, the rest of the network continues to operate.

  • Mesh - In this topology, every device is connected to every other device on the network. The main advantage is that it is extremely reliable as there is no single point of failure. The main disadvantage is that it is very expensive to set up.

Module 3: Advanced Network Concepts

Virtual Local Area Network (VLAN)

A virtual local area network (VLAN) is a logical network that can group together devices from different physical networks. VLANs can be used to improve security by isolating network traffic and preventing attacks. Administration of the environment is simplified, as the VLANs can be reconfigured when individuals change their physical location or need access to different services.

VLAN Security

VLANs do not guarantee a network’s security. At first glance, it may seem that traffic cannot be intercepted because communication within a VLAN is restricted to member devices. However, there are attacks that allow a malicious user to see traffic from other VLANs (so-called VLAN hopping). The VLAN technology is only one tool that can improve the overall security of the network environment.

Virtual Private Network (VPN)

A virtual private network (VPN) is not necessarily an encrypted tunnel. It is simply a point-to-point connection between two hosts that allows them to communicate.

Secure communications can, of course, be provided by the VPN, but only if the security protocols have been selected and correctly configured to provide a trusted path over an untrusted network, such as the internet. Remote users employ VPNs to access their organization’s network, and depending on the VPN’s implementation, they may have most of the same resources available to them as if they were physically at the office.
