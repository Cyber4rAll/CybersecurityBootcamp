Prerequisites & Structure

📌 Prerequisites Disclaimer

  • Basic computer literacy is required.
  • Familiarity with networking concepts (IP addresses, TCP/IP, DNS) is helpful but not mandatory.
  • No prior cybersecurity experience required—this bootcamp is beginner-friendly but intensive.

Bootcamp Structure

  • Total Duration: 9 Weeks
  • Format: Instructor-led (live sessions) + self-paced study materials
  • Domains Covered: 5 domains of ISC² CC + 2 weeks Ethical Hacking fundamentals

📘 (ISC)² CC Domains

1. Security Principles

2. Business Continuity, Disaster Recovery & Incident Response

3. Access Controls Concepts

4. Network Security

5. Security Operations

🗓 Weekly Breakdown: Weeks 1–7

Weeks 1-3

  • Week 1: Domain 1 – Security Principles
  • Week 2: Domain 2 – BCDR & Incident Response
  • Week 3: Domain 3 – Access Control Concepts

Weeks 4-5

  • Week 4: Domain 4 – Network Security (Part 1)
  • Week 5: Domain 4 – Network Security (Part 2)

Weeks 6-7

  • Week 6: Domain 5 – Security Operations (Part 1)
  • Week 7: Domain 5 – Security Operations (Part 2)

🗓 Weekly Breakdown: Weeks 8–9

Week 8: Introduction to Ethical Hacking

  • Hacking methodology (Recon → Exploitation → Post-exploitation)
  • Lab setup (Kali Linux, Metasploitable, Wireshark basics)
  • Recon & scanning techniques (Nmap, Whois, Shodan)

Week 9: Hands-On Ethical Hacking

  • Web application security (OWASP Top 10 overview)
  • Password attacks & defenses
  • Basic exploitation demo (SQL injection, XSS)
  • Ethical considerations + career pathways (CEH, OSCP, PenTest+)

⏰ Cohort Structures

🅰️ Weekday Cohort

  • Schedule: Mon, Wed, Fri
  • Duration: 2 hours per session (6 hours total per week)
  • Format:
    • Mon → New topics + lecture
    • Wed → Deep dive + labs/case studies
    • Fri → Recap + practice questions

🅱️ Weekend Cohort

  • Schedule: Sat & Sun
  • Duration: 3 hours per session (6 hours total per week)
  • Format:
    • Sat → Lecture + labs
    • Sun → Recap + discussion + practice tests
